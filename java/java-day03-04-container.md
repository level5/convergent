### List， Set， Map
* 三者的区别
  - List,顺序保存元素，
    - ArrayList， LinkedList，通过这两个List可以演示一下大数据量下，删除第一个元素时所花的时间。巩固前面关于顺序和链表的差别
  - Set，无序，不能有重复元素
  - Map，键值对，无序，不能有重复的key。使用键来查找。
* Set和Map是常量时间查找和删除
